# Travel Company Management System

Console-based Travel Management System using Java, JDBC, and MySQL.

## Project Description
This is a Java-based console application that manages a Travel Company system using JDBC and MySQL. It handles bus details, passenger details, and booking operations with proper database connectivity.

## Technologies Used
- Java
- JDBC
- MySQL
- IntelliJ IDEA / Eclipse

## Features
- Add Bus Details
- Add Passenger Details
- Book Tickets
- View Records
- Update/Delete Records

## Database
Database Name: travel_company  
Tables Used:
- bus
- passenger
- booking

## How to Run
1. Import project in IDE  
2. Create MySQL database (travel_company)  
3. Run SQL file to create tables  
4. Update database username & password in code  
5. Run main Java class  

## Note
- Make sure MySQL server is running  
- JDBC connector must be added in project
